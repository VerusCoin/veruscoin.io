Start Mining VRSC Now!

Anyone with a capable computer can mine Verus Coin (VRSC)!
  • No specialized hardware is required!
  • Mine with either CPU or GPU power.
  • Mine directly from the Verus Wallet or use the CLI Miner.
However you mine VRSC, you'll need a Verus Coin address, which you can create using the wallet in either Lite or Native mode. If you need any help, check our How-To and FAQ section, or join our official Discord and reach out to the community support channel. Happy mining!
